Welcome to the class roster execise!

For this exercise you will be making a very simple web app that contains an array of student names.The website will prompt you for four possible comands: add,remove,display,and quit. Here is what each command should do:

  • add: Add this name to the student to the array of student name.
  • remove: remove name from the array.
  • display: print out roster
  • quit: willend the loop of prompts.
